# WiGuard

## Description

un script permettant de bloquer des individus inconnus sur votre réseau en filtrant des addresses mac.

## Objectif

- Protéger ton réseau des intrus
- Respet du vie privée

## Comment fonctionne le script ?

il suffit de faire deux choses:

1. Télécharger le fichier ou directement cloner le répository github

```bash
git clone repo shinra_tensei
cd shinra_tensei
```

2. Gérer le droit d'exécution du script avec la commande `chmod` puis l'exécuter

```bash
sudo chmod 777 script.sh
./script.sh
```

**Rémarque**:

- Pour l'instant, je n'avais pas l'ocasion de le tester susr **mac** ou **windows** mais juste pour **linux**
- Vous devez connaitre tous la liste d'addresse mac des pc qui peuvent utiliser le réseaux
- Vous devez aussi indiquer le nombre de chanel du réseau dans le script
